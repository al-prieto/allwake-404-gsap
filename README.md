## Overview

The project started from a visual walkthrough created in **Figma**, followed by a motion prototype in **Jitter** to study timing and flow.  
From there, the experience was rebuilt from scratch using **HTML, CSS, and vanilla JavaScript**, adapting the animation logic for the web.

The original character was replaced with a **custom alpaca video generated using AI**, integrated directly into the layout and animation flow.

---

## Key Features

- GSAP timeline-based entrance animation
- Character-level text animation by programmatically splitting text into spans
- Coordinated stagger between background typography, copy, and CTA
- Subtle environmental motion (cloud drift)
- Responsive animation behavior for desktop and mobile
- Clean, modular CSS with design tokens for colors and typography

## Purpose

This project serves as a **micro interaction demo** to showcase:

- Motion design translation from tools like Figma and Jitter into real code
- GSAP timeline control and sequencing
- Attention to detail in UI animation and perceived quality
- Front-end animation skills relevant to modern Webflow / JavaScript-focused roles
